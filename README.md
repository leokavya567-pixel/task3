# Task 3 - SQL for Data Analysis

## Objective

To analyze ecommerce data using SQL queries and understand data manipulation techniques.

## Tools Used

* MySQL Workbench
* MySQL Database

## Dataset

Sample Ecommerce Orders Dataset

## SQL Concepts Covered

* SELECT
* WHERE
* ORDER BY
* GROUP BY
* Aggregate Functions (SUM, AVG, MAX, MIN)
* Subqueries
* Views
* Indexes

## Queries Performed

1. Display all records
2. Filter records using WHERE
3. Sort data using ORDER BY
4. Group data using GROUP BY
5. Calculate aggregates
6. Use subqueries
7. Create and query views
8. Create indexes

## Outcome

Learned how to retrieve, filter, sort, aggregate, and analyze structured data using SQL.
